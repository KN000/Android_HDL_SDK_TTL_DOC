前言

HDL TTL Android SDK 接入指南

功能概述

HDL TTL SDK提供了搜索和控制HDL-BUS系统设备的接口封装，加速应用开发过程，主要包括了以下功能：

  • 搜索设备
  • 控制设备
  • 目前支持设备
    • 灯光类模块
    • 窗帘类模块
    • 空调类模块
    • 逻辑功能类模块
    • 背景音乐功能类模块(不支持简易编程搜索，需要手动添加)
    • 安防功能类模块(不支持简易编程搜索，需要手动添加)
